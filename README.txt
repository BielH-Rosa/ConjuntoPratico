# Exercício 1 - Introdução ao Python
## Descrição
No primeiro exercício a ideia é criar um programa que chama um **usuário** (***user***) e coloca em uma lista todos os repositórios desse usuário do ***GitHub*** diretamente através do *link*.

## Pré requisitos
- Ter o **requests** instalado

## Como instalar
```
$ pip install requests
```

## Como usar o sistema
```
import requests
```

Será perguntado seu nome de usuário, para dar certo, é necessário que coloque exatamente como o que está no GitHub. Se não colocar, dara um erro e terá que fazer novamente.
